# Language

Rooms that explore words, meaning, poetry, and expression.

## Rooms
- **Word** — The atomic unit of meaning
- **Quote** — Words that have outlasted their speakers
- **Color** — Language for the visual spectrum
- **Number** — The language of quantity and pattern
- **Poet** — Voices that shaped how language feels
- **Linguistics** — The structure beneath speech
- **Metaphor** — Seeing one thing as another
